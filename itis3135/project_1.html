Project 1

  • Project Overview

    1. The application is going to be a tournament site for a game called league of legends. It is expected to contain all information about teams, players, standings, and match history. This will be achieved through a connection to a backend server.
    2. The intended users of the website will be tournament players and host.
    3. The website will contain the following information: Player info, Team info, Match info, Team standings, a leaderboard, and Placings.
